Лето 2018

разработка второго коммерческого проекта

TradeBots | retarcorp

Front-end & Back-end разработка

TradeBots - веб-платформа, предоставляющая ботов для трейдинга на криптобирже Binance.

В данном проекте мной были разработаны - RESTful сервер, боты и веб-интерфейс.

Для работы ботов, необходимо было активно использовать Binance API для получение информации о текущих ценах на рынке и работы с пользовательскими ордерами. Также пришлось парсить сигналы tradingview и привязывать оплату криптовалютой через bitaps.

Сам интерфейс позволял создовать и испльзовать ботов двух видов - автоматических и ручных. Для работы с ручными ботами необходимо произвести настройку (ввести лимиты, начальные суммы, пару валют и кол-во страовочных ордеров) и запустить его, после чего, он начинал выставлять оредера на биржу и реагировать на текущее положение. Для автоматических - ввести начальные суммы, лимиты, набор пар, тип сигналов и запустить, далее бот начинал следить за сигналами tradingview, сравнивал с введенными, и если было совпадение, то открывал сделки на бирже.

Этот проект предоставил бесценный опыт работы с асинхронными функциями, внешними API и опыт серверной разработки.
